# Crossbar Array Measurement Tracker

Interactive web application for tracking ferroelectric diode measurements across crossbar arrays.

## Features

- Support for 8x8 and 128x128 (2kB) crossbar arrays
- Click-based measurement tracking:
  - 1 click = Green (successful)
  - 2 clicks = Red (failed)
  - 3 clicks = Yellow (misaligned)
  - 4 clicks = Reset to unmeasured
- Multiple entry management with persistent storage
- Real-time statistics and percentages
- JSON export for data analysis
- Browser-based localStorage persistence

## Usage

Simply open the website and start creating measurement entries for your crossbar arrays.
